How to use publican_creators

A graphical user interface for publican and fo2xxx

This program works on top of Redhat's publican. It helps you create new projects and replaces the default content with your own information. This is done by publican_creators using a graphical project developer, a build script and a graphical revision editor. In addition to the output formats supported by publican, publican_creators supports the formats docx, odt, rtf and wml (using the XFC FO- converter).
